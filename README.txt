Use the clipmanager to easily text pasting (run Clipmanager.exe) and setting if you want edit something (run Setting.exe)
program automatically processes all copied texts when it is running

hotkeys:
alt+v --- show the clipmanager (after running)
when program is shown:
enter --- paste current clip (program is sent to tray after this but you can edit it within setting)
delete --- delete current clip
arrows (up, down) --- move between clips
escape --- send program to tray

Click close button when program is shown to close it
